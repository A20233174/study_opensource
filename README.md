리눅스 명령어 정리 (top, ps, jobs, kill)
===============
## 1. top

top 명령어는 시스템의 현재 상태와 실행 중인 프로세스에 대한 정보를 실시간으로 제공하는 명령어이다.

CPU 사용량, 메모리 사용량, 프로세스 목록 및 관련 정보, 로드 평균 등을 확인할 수 있다.

시스템 모니터링, 성능 분석을 하는 명령어 이므로 윈도우의 '**작업 관리자**'와 비슷한 기능을 한다고 볼 수 있다.
